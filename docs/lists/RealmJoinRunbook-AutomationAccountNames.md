# Automation Account Runbook Names

This document lists all runbook names as they appear in the Azure Automation Account after synchronization.
The naming format is: `rjgit-<folder>_<subfolder>_<runbookname>` (all lowercase, without .ps1 extension).

```json
{
    "rjgit-device_general_change-grouptag",
    "rjgit-device_general_check-updatable-assets",
    "rjgit-device_security_enable-or-disable-device",
    "rjgit-device_security_isolate-or-release-device",
    "rjgit-group_general_add-or-remove-nested-group",
    "rjgit-group_general_add-or-remove-owner",
    "rjgit-group_general_add-or-remove-user",
    "rjgit-group_mail_enable-or-disable-external-mail",
    "rjgit-group_mail_show-or-hide-in-address-book",
    "rjgit-user_general_assign-groups-by-template",
    "rjgit-user_general_assign-or-unassign-license",
    "rjgit-user_general_assign-windows365",
    "rjgit-user_general_list-group-memberships",
    "rjgit-user_mail_add-or-remove-email-address",
    "rjgit-user_mail_assign-owa-mailbox-policy",
    "rjgit-user_mail_convert-to-shared-mailbox"
}
```
